O painel salva as edições como commits no repositório, então a "senha" é um token de acesso do GitHub — só você o tem, e ele fica apenas neste navegador.

Crie em github.com → Settings → Developer settings → Fine-grained tokens. Repositório: Portif-lio_de_aplicativo · Permissão: Contents → Read and write.
